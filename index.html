History

  • Originally named JavaScript
  • Netscape submitted JavaScript to ECMA International for Standardization
  • Result as a new language standard: ECMAScript
  • ECMAScript is a standard and JavaScript is an implementation of this standard

History

  • ES1: June 1997

  • ES4: Abandoned
  • ES5: Dec. 2009
  • ES6 || ES 2015: Jun. 2015
  • ES2016: June 2016
  • ES2017: June 2017
  • ES.next: dynamic reference to the next version
